Atividade

É preciso que você desenvolva um pipeline de automação utilizando o GitHub Actions, simulando um processo de CI/CD adotado em empresas que aplicam práticas DevOps. O pipeline será responsável por validar arquivos, rodar testes simples, gerar artefatos e publicar automaticamente uma página estática no GitHub Pages. O objetivo principal é entender, na prática, como fluxos automatizados garantem qualidade, rapidez e padronização no desenvolvimento de software.
